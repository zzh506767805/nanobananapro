# Nano Banana Pro 项目

使用 Google 最新的 Nano Banana Pro (gemini-3-pro-image-preview) 模型生成和编辑图片。

## 技术栈
- Python 3.9+
- google-genai SDK
- 模型: gemini-3-pro-image-preview

## 快速开始
1. 复制 .env.example 为 .env，填入 API Key
2. pip install -r requirements.txt
3. python main.py generate "描述"

- 4K 图片: $0.24/张
